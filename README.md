# Coassemble Headless Demo

This is a demo of how to integrate a third-party platform with Coassemble's headless API.

The main files to take note of are in `/server/api`â€”these detail how to authenticate with Coassemble and the main operations you can perform.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
